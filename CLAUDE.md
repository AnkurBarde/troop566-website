# Troop 566 Website

## Project Overview
Static website for Boy Scout Troop 566, a troop based in Saratoga, CA active since 1973.

## Structure
- `index.html` — single-page site with all content and inline CSS

## Design
- **Primary color:** `#b50000` (red)
- **Dark accent:** `#1a1a2e` (navy)
- **Font:** Segoe UI
- Sections: Nav, Hero, Stats, About, Activities, CTA, Footer

## Key Info
- Meeting: Every Tuesday at 7:00 PM at Menlo Church
- Contact: info@troop566.org / (555) 123-4567
- Stats: 80+ scouts, 30+ Eagle Scouts, 25+ events/year, 50+ years active

## Development Notes
- No build system — plain HTML/CSS, no frameworks or dependencies
- All styles are inline in `<style>` within `index.html`
- Hero background image sourced from Unsplash
